## Major Upgrades Delivered During Day 16

### Document Search by Meaning

Instead of exact filenames, RohitOS now understands context via multi-tier matching (strict alias → strict fuzzy → multi-substring → low-confidence suggestions). 

### Safe Mathematical Execution

Developed a highly secure, restricted `ast.parse` tree evaluator to locally resolve math expressions (e.g., "calculate 1049 x 246") without spending AI quota or invoking dangerous `eval()` functions.

### Perfected Fallback Priorities

Documents are evaluated first for fuzzy/keyword matches, then applications, preventing apps from hijacking document requests.
